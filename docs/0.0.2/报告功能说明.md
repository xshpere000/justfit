# 报告功能说明

JustFit 提供专业的资源评估报告功能，帮助您直观了解云平台资源状况。

## 报告类型

### Excel 报告

**特点**：
- 包含所有资源明细数据
- 支持二次编辑和数据分析
- 适合技术团队深入分析

**内容**：
- **概览**：资源总量统计
- **集群信息**：各集群资源分布
- **主机信息**：物理主机配置与状态
- **虚拟机列表**：所有 VM 详细信息
- **僵尸 VM**：低使用率虚拟机详情
- **规格调整**：资源优化建议
- **潮汐检测**：周期性使用模式
- **健康评分**：平台整体健康度

### PDF 报告

**特点**：
- 图文并茂，易于阅读
- 包含统计图表
- 适合向管理层汇报

**内容**：
- 资源概览统计
- 集群与主机分布
- 分析结果汇总
- 可视化图表（饼图、柱状图等）
- 优化建议清单

## 报告生成方式

### 按任务生成

在任务详情页面，点击「生成报告」：
- 自动使用该任务的数据
- 包含采集的资源和分析结果
- 支持生成多份报告

### 全局报告

在报告管理页面：
- 使用所有历史数据
- 跨任务的资源统计
- 适合整体资源盘点

## 报告内容说明

### 僵尸 VM 检测

**含义**：长期低使用率的虚拟机，可能造成资源浪费

**判断依据**：
- CPU 使用率低于阈值（如 5%）
- 内存使用率低于阈值（如 10%）
- 磁盘 I/O 和网络流量低于阈值
- 持续时间超过设定天数（如 14 天）

**建议处理**：
- 关机不重要的测试 VM
- 删除长期闲置的 VM
- 合并低负载 VM

### 规格调整

**含义**：虚拟机资源配置不合理的情况

**类型**：
- **扩容**：资源不足，建议增加配置
- **缩容**：资源过剩，建议降低配置
- **大幅扩容**：严重不足，需要紧急扩容
- **大幅缩容**：严重过剩，可大幅降低配置

**风险等级**：
- 低风险：建议可信，可直接执行
- 中风险：需要评估影响
- 高风险：建议谨慎，需详细分析

### 潮汐模式检测

**含义**：虚拟机使用呈现周期性规律

**模式类型**：
- **周周期**：工作日与周末使用差异明显
- **日周期**：白天与夜间使用差异明显
- **无模式**：使用较为平稳

**应用价值**：
- 资源分时复用
- 节能调度
- 容量规划

### 健康评分

**评估维度**：
- **资源均衡度**：集群间资源分配是否均衡
- **超配风险**：资源超分配程度
- **热点集中度**：过载是否过度集中

**评分等级**：
- 90-100 分：优秀
- 75-89 分：良好
- 60-74 分：一般
- 0-59 分：需要改进

## 使用建议

1. **定期生成报告**：建议每月或每季度生成一次，跟踪资源变化
2. **对比分析**：保存不同时期的报告，对比资源变化趋势
3. **结合评估模式**：使用不同的评估模式生成报告，获得不同视角
4. **导出数据**：Excel 报告可用于进一步的数据分析

## 注意事项

- 报告基于已采集的数据生成
- 建议先完成数据采集和分析再生成报告
- PDF 报告中的图表需要足够的样本数据才能生成
